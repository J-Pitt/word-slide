# 🚀 Test Cheat Sheet

## Quick Commands

```bash
# Run all tests
npm test -- --run

# Watch mode (auto-run on save)
npm run test:watch

# Coverage report
npm run test:coverage

# Before deploying
npm run pre-deploy
```

## Daily Workflow

```bash
# Morning
git pull && npm test -- --run

# During dev
npm run test:watch

# Before commit
npm test -- --run && git add . && git commit -m "..."

# Before push
npm run pre-deploy && git push
```

## When Tests Fail

1. Read the error message
2. Run just that test: `npm test -- -t "test name" --run`
3. Fix the issue
4. Re-run all tests
5. All green? ✅ Continue

## Adding New Feature

1. ✅ Baseline: `npm test -- --run` (all pass)
2. 📝 Write test for new feature (should fail)
3. 💻 Implement feature
4. ✅ Run test (should pass)
5. ✅ Run all tests (should all pass)
6. 🚀 Deploy!

## Protection

**Before ANY commit:**
```bash
npm test -- --run
```

**If tests fail → DON'T commit!**

Your game is protected! 🛡️
